Customer Churn Prediction
 Project Overview
This project focuses on analyzing telecom customer data to understand customer churn patterns and identify factors associated with customers leaving the service.
The project was completed as part of my Data Analytics internship at SaiKet Systems.

 Objectives
- Understand the customer churn dataset
- Perform data cleaning and preprocessing
- Identify and handle missing values
- Remove duplicate records
- Perform Exploratory Data Analysis (EDA)
- Analyze customer behaviour and churn patterns
- Visualize important customer and service-related features

 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

 Dataset
The Telco Customer Churn dataset contains customer demographic information, service details, account information, monthly charges, total charges, and customer churn status.

 Project Workflow
1. Dataset loading and understanding
2. Column name standardization
3. Missing value analysis
4. Duplicate record analysis
5. Data cleaning and preprocessing
6. Exploratory Data Analysis
7. Customer behaviour analysis
8. Data visualization
9. Churn pattern analysis

 Key Analysis
The project analyzes important customer attributes such as:
- Senior citizen status
- Tenure
- Internet service
- Contract type
- Payment method
- Monthly charges
- Total charges
- Churn status

 Repository Files
- Customer_Churn_Prediction.ipynb - Python analysis and visualizations
- Telco_Customer_Churn.csv - Dataset used for analysis
